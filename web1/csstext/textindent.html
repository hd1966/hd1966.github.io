Text indent and font variant

The text-indent property specifies indentation of the first line in a text-block. In this text text-indent is set to 40px.

Lorem ipsum dolor sit amet, consectetur
adipiscing elit. Integer vitae volutpat sapien,
eu suscipit lorem. Suspendisse lacinia
nibh vel nibh gravida accumsan vitae eu tellus.

The font-variant proiperty specifies whether text will be displayed in a small-caps font.The font-variant in this text is set to small-caps.

Quisque tempor tempus erat, sed auctor tellus commodo ut.
Suspendisse sed nisl convallis, imperdiet leo at, fringilla dui.
Etiam auctor aliquet scelerisque.
